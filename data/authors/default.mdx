## muzzledskr について

muzzledskr（マズルド・スクレ）は、フリーランスのインフラエンジニア。AWS / SRE / クラウドインフラを中心に技術記事を、note などで動機の観察エッセイを書いている。

書いているテーマは2つ。

**1. 制約された現場での技術**

踏み台サーバ越しの作業環境、業務PCに自由なツールを入れられない現場、セキュリティポリシーで Postman が使えない状況など、エンタープライズや SIer 現場のリアルな制約の中で、vim / curl / bash などの古典ツールを再発見しながら書く。

**2. 動機の観察**

資格を取った時の動機、業務で Java を学んだ動機、それが内発化していった経緯、最終的に Java から離れる選択。自分や他者の動機の変化を時系列で言語化する。
